# Continuous Learning Skill

Self-improvement from sessions.

## When to Use
- Learning from errors
- Improving task execution
- Adapting to project patterns
- Building knowledge base

## What_You_MUST_Do>
1. REVIEW session summaries for patterns
2. CAPTURE non-obvious insights with context
3. IDENTIFY recurring issues
4. DETECT coding patterns and team preferences
5. ADAPT to codebase style

## What_You_MUST_NOT_Do>
1. DO NOT capture obvious or trivial insights
2. DO NOT forget to tag with context
3. DO NOT skip pattern detection

## What This Skill Does

### Pattern Detection
- Identify recurring issues
- Detect coding patterns
- Learn team preferences
- Adapt to codebase style

### Knowledge Capture
```typescript
await captureInsight({
  topic: "auth-pattern",
  pattern: "jwt-with-refresh"
})
```

## Best Practices
1. Review session summaries
2. Capture non-obvious insights
3. Tag with context
